SKY66123 - High-Efficiency PA Module

Part Number: SKY66123 Status: Active

Description

The SKY66123 is a highly integrated front-end module (FEM) designed for LTE handsets and data cards. This compact module integrates all RF functions including power amplification, harmonic suppression, and antenna switching into a single 6-pin MGA package.

Specifications

Frequency Range 698-960 MHz
Supply Voltage 2.5-4.2V
Output Power +23 dBm
Gain 33 dB
Efficiency 65%
Package 6-pin MGA
